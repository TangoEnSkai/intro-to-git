# Section 4 - Branches

Topics are:

* Git Branches
  * Listing all branches
    * `git branch`
    * `git branch -a`
  * Adding a branches
    * `git checkout -b <branch-name>`
      * be aware that there might be conventions for branch names depending on your team
  * Changing branches (e.g. change to `master`)
    * `git checkout master`
  * Merging a branch
  * Removing a branch
